How to use:
①Put faith.ilp.lua into the ilua folder and run it again to automatically create a configuration file
②Go to the en_US folder to find config.json 
③Then put dkjson.lua and config.json into the faith folder under the plugins folder 

/faith - open the title panel
/faith help - View help
/faith list - View titles you already own
/faith view - View the current title
